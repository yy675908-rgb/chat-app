# 产品边界 v0.4

## 核心体验

角色不是每次打开后才临时出现的回答机器。它应记得共同经历、感知时间经过，并能以“正在想”“撤回改口”“主动想念”等状态呈现自己的连续存在。

## 当前版本

- 内置一个可编辑角色“林”，先把聊天手感和关系连续性打磨好。
- 聊天记录保存在本机。
- 共同记忆与普通聊天记录分开，并会被加入模型上下文。
- 供应商、模型、角色和会话分层；聊天页可以快速切换模型。
- 支持 OpenAI 兼容接口与 Anthropic 原生接口，回复以流式呈现。
- 支持多会话、停止生成、重新生成、复制与 Markdown 显示。
- API Key 使用系统加密存储，不写入普通配置或聊天记录。

## 后续能力

- 角色卡、世界书、多角色
- 共同日记、分享聊天链接、角色语音
- 主动消息和系统通知
- 悬浮小窗
- 经明确授权后调用搜索、文件与部分 Android 系统能力
